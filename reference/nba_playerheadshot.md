# **Get NBA Stats API Player Head-shot**

**Get NBA Stats API Player Head-shot**

**Get NBA Stats API Player Head-shot**

## Usage

``` r
nba_playerheadshot(player_id = "2544", ...)
```

## Arguments

- player_id:

  Player ID

- ...:

  Additional arguments passed to an underlying function like httr.

## Value

Returns a url of the png for the player_id selected
