# **Men's College Basketball KenPom Teams Dictionary** Team link KenPom reference lookup for the package

**Men's College Basketball KenPom Teams Dictionary** Team link KenPom
reference lookup for the package

## Usage

``` r
teams_links
```

## Format

A data frame with 357 rows and 6 variables:\\

- `Team`:

  character.

- `Team.link`:

  character.

- `team.link.ref`:

  character.

- `Year`:

  double.

- `Conf`:

  character.

- `Conf.link`:

  character.

- `conf.link.ref`:

  character.
